# 💻 Portfólio - Catharina Alves Bonella

Bem-vindo ao meu portfólio! 🚀
Este projeto foi desenvolvido para apresentar minhas habilidades e projetos como desenvolvedora web.

---

## ✨ Sobre o projeto

Portfólio moderno, responsivo e interativo, com foco em:

* Design limpo e organizado
* Experiência do usuário (UX/UI)
* Navegação fluida entre seções
* Estrutura simples e funcional

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* TailwindCSS
* Vite

---

## 📂 Funcionalidades

* Página única (Single Page)
* Navegação suave entre seções
* Seção "Sobre mim"
* Exibição de habilidades com ícones interativos
* Cards de projetos
* Seção de contato com links externos
* Layout responsivo
